PDFTool - 从压缩包到独立 exe 完整步骤
========================================

【最简单】双击一个文件
------------------------------------------
解压 zip 后，双击：

  PDFTool.bat

首次约 3~5 分钟（自动装 Python + 生成 exe + 启动）。
以后再次双击 PDFTool.bat，会直接打开程序。

最终独立程序在：

  dist\PDFTool.exe

可单独拷贝 dist\PDFTool.exe 给他人，对方无需 Python、无需再打包。

========================================
【说明】为什么不能直接给一个 exe？
========================================

Windows 的 .exe 必须在 Windows 系统上编译。
Mac 上无法直接生成 Windows 程序。

因此有两种方式：
1. 在 Windows 上双击 PDFTool.bat，自动出 exe（只需一次）
2. 用 GitHub Actions 云端打包后下载 exe（见 README）

========================================
【准备】
- Windows 10 / 11（64 位）
- 能联网（首次约 3~5 分钟）
- 压缩包：
  PDFTool/release/PDFTool-Windows.zip

========================================
【手动步骤】（与 PDFTool.bat 等价）
========================================

1. 解压 zip
2. 安装Python环境.bat
3. 安装并打包.bat
4. 使用 dist\PDFTool.exe

========================================
【可选】Word 转 PDF
========================================

需安装 LibreOffice 或 Microsoft Word，其他功能不受影响。

========================================
常见问题
========================================

Q: 提示「python311 不是命令」？
A: 不要手动输入 python311。请双击 PDFTool.bat 或 检查环境.bat

Q: 只想把程序发给别人？
A: 打包成功后，只发 dist\PDFTool.exe 即可（约 50~80MB）

Q: 杀毒软件报毒？
A: PyInstaller 常见误报，添加信任即可
