Windows Python 运行环境包
========================

本目录包含 PDFTool 在 Windows 上运行所需的 Python 安装包（无需单独上网下载 Python）。

文件说明
--------
- python-3.11.9-amd64.exe   官方 Python 3.11.9 安装程序（64 位，含 tkinter）
- get-pip.py                pip 备用脚本（一般不需要，安装包已含 pip）

使用步骤（在项目根目录 PDFTool 下）
----------------------------------
1. 双击「安装Python环境.bat」  → 安装到 runtime\python311\
2. 双击「安装依赖并运行.bat」  → 安装 pypdf / pymupdf 等并启动工具

说明
----
- 环境安装在项目内 runtime\python311\，不修改系统 PATH
- 首次安装依赖需联网（pip 下载库）
- 若需独立 exe，仍请使用「安装并打包.bat」
