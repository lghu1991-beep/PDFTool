PDFTool 在 Windows 10 上“双击没反应”排查说明
============================================

你的系统：Windows 10 专业版 21H1（内部版本 19043）
该版本可以运行本工具，但需要重新打包一版兼容性更好的 exe。

一、先做这 3 步（在出问题的电脑上）
------------------------------------
1. 右键 PDFTool.exe → 属性 → 若底部有“解除锁定”，请勾选后确定
2. Windows 安全中心 → 病毒和威胁防护 → 保护历史记录
   看是否拦截了 PDFTool.exe，若有则“允许”
3. 安装运行库（很多 PyInstaller 程序需要）：
   “Microsoft Visual C++ 2015-2022 可再发行组件（x64）”
   微软官网搜索 VC++ redistributable x64 下载安装

二、用调试版看真实报错（推荐）
------------------------------------
在能打包的 Windows 上执行：build-debug.bat
把 dist\PDFTool-debug.exe 拷到问题电脑，双击运行。

会弹出黑色命令行窗口 + 程序界面。
若仍失败，窗口里会有英文报错，请截图发维护人员。

同时查看日志文件：
  %LOCALAPPDATA%\PDFTool\startup.log
（资源管理器地址栏输入 %LOCALAPPDATA%\PDFTool）

三、本次代码已做的兼容性修复
------------------------------------
- 关闭 UPX 压缩（部分 Win10 + 杀毒软件会导致 exe 无法启动）
- 启动失败时写入 startup.log 并弹窗提示
- 修复密码框在 Win10 上可能不显示（先 update 再弹窗）
- MuPDF 强压缩在打包环境下加载 DLL 的路径修复

四、重新打包正式版
------------------------------------
在 Windows 上进入 PDFTool 目录，双击：安装并打包.bat
用新生成的 dist\PDFTool.exe 替换旧文件。

五、仍不行时的备选
------------------------------------
不依赖 exe，直接运行：
  安装Python环境.bat
  安装依赖并运行.bat
需要本机有网络安装依赖一次。
