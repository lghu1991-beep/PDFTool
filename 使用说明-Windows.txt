PDFTool - Windows 使用说明
【方式一】生成独立 exe（推荐，无需 Python）
------------------------------------------
1. 将整个文件夹复制到 Windows 电脑
2. 双击运行：安装并打包.bat
   （或已装 Python 时直接双击 build.bat）
3. 等待 2~5 分钟，完成后得到：
   dist\PDFTool.exe
4. 双击 PDFTool.exe 即可使用，可拷贝到任意目录

【方式二】开发模式（需已安装 Python）
------------------------------------------
双击 run-dev.bat 直接运行，不生成 exe。

【系统要求】
------------------------------------------
- Windows 10 / 11（64 位）
- 打包时需要：Python 3.10+（含 tkinter）
- Word 转 PDF 需要额外安装：
  LibreOffice（推荐，免费）
  https://www.libreoffice.org/download/

【功能】
------------------------------------------
- Word 转 PDF（doc/docx/wps/rtf 等）
- 文字水印 / 图片水印
- PDF 压缩（轻 / 中 / 强）
- PDF 合并 / 拆分

【常见问题】
------------------------------------------
Q: 杀毒软件报毒？
A: PyInstaller 打包的 exe 可能被误报，添加信任即可。

Q: Word 转 PDF 失败？
A: 安装 LibreOffice 后重试。

Q: 强压缩效果不明显？
A: 纯文本 PDF 用「中」即可；扫描件/大图 PDF 用「强」。

Q: macOS 能否打包 Windows exe？
A: 不能，必须在 Windows 上运行 build.bat。
